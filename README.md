# Memory Matchup - Test Your Memory Skills!
<img width="2048" alt="memory-matchup-cover" src="https://github.com/ssindher11/MemoryMatchup/assets/34216257/f7db797d-7922-480f-aea6-b46c691ec92b">

Memory Matchup is an addictive and challenging Android game built with Jetpack Compose. Put your memory skills to the test as you try to match tiles in as few flips as possible. With multiple difficulty levels, star ratings, and exciting background music, it's the perfect game to sharpen your mind and have fun!

## Features
### Multiple Difficulty Levels
Choose your challenge! Memory Matchup offers three difficulty levels: easy, medium, and hard. Start with the one that suits your skill level or dive right into the toughest puzzles.

### Star Ratings
Earn star ratings based on your performance. The fewer flips and less time you take to complete a level, the more stars you'll receive. Can you earn that elusive 3-star rating on every level?

### Engaging Background Music
Immerse yourself in the game with engaging background music that adds to the overall experience.

### Offline Play
No internet connection? No problem! Memory Matchup is fully offline, allowing you to enjoy it wherever and whenever you like.
